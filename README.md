# AI Communication Coach

## Overview
The AI Communication Coach is an intelligent web application designed to help users improve their communication skills. By having a 5-minute live conversation with an AI agent, users receive an in-depth, structured report assessing their performance, highlighting strengths and weaknesses, and offering actionable insights to help them grow.

## Project Structure
This project is divided into two main services:
- [**Frontend**](./frontend/README.md): A Next.js UI for the user to interact with the AI assistant.
- [**Backend**](./backend/README.md): A Spring Boot server handling real-time WebSockets, AWS Bedrock integration, and media processing.

## Getting Started

### Prerequisites
- **Node.js** (v20+ recommended)
- **Java Development Kit (JDK)**
- **Apache Maven**
- **AWS Account** with access to AWS Bedrock and S3 (ensure your AWS credentials are appropriately configured).

### General Installation
1. **Clone the Repository**
   ```bash
   git clone https://github.com/parmar-mohit/AI-Communication-Gym
   cd AI-Communication-Gym
   ```

2. Follow the detailed setup instructions for the **Frontend** and **Backend** in their respective directories:
   - [Frontend Setup Instructions](./frontend/README.md)
   - [Backend Setup Instructions](./backend/README.md)
